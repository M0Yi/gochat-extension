连接方式

两种模式
灵活部署

根据你的网络环境和安全需求，选择最适合的连接模式。

Relay 模式

所有消息通过 GoChat 服务器中转。适合需要服务器端管理和多端同步的场景。

Direct 模式

OpenClaw 插件与客户端直接建立 WebSocket 连接。适合内网环境或对延迟敏感的场景。
